#### Usage deriv_test ####

## Compilation

In the `C-or-Cpp` directory change the working directory to `deriv_test`, `make` or `make deriv_test` will generate an executable `deriv_test` from the source files.

## Running

Once compiled, the `deriv_test` executable can be run directly from terminal: `./deriv_test`

This produces the following:
- 261023938_CA_2_Deriv_Test.dat : Contains the values of x, f(x), f'(x), and f"(x) for f(x) = sin(x). These results are plotted in plotted_outputs_for_deriv_test.png.

## Cleaning

To safely remove the `deriv_test` executable, run `make clean` in terminal.
